## 🚀 QUICK START

### Prerequisites
- Node.js 18+
- Redis (for streaming)
- PostgreSQL (for ML models)
- Kubernetes (for multi-region)

### Installation
```bash
# Install dependencies
npm install

# Start services
docker-compose up -d

# Initialize ML models
npm run ml:train

# Start application
npm run start:prod
```
