## 🚀 QUICK START

### Prerequisites
- Node.js 18+
- Redis (for caching)
- PostgreSQL (for audit trails)
- Docker (for containerization)

### Installation
```bash
# Install dependencies
npm install

# Start services
docker-compose up -d

# Run migrations
npm run migrate

# Start application
npm run start:prod
```
